# NGS_Seq_Scripts

## Sara Nicholson

## Programs
**scRNA_Pipeline_Basic.Rmd** - Details the steps of Single-Cell-Sequencing Analysis from expression matrix to Filtering/Normalization, Integration, Doublet Removal, Dimensional Reduction, Clustering, Modeling, Differential Expression, Trajectory Analysis and many different visualizations.\
**Finding_HIV_Tropism.py** - Labeling HIV Sequence as X4 or R5 (X4 or R5-derived), returns dictionary of annotations. \
**app.R** - RShiny Appy created for visualization of Single-Cell Expression in Lung or Bone Marrow Samples for HIV-Infected and Mock samples.\
**helpers.R** - Helper Functions File associated with RShiny app detailed above.\
